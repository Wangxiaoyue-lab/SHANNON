# DeepCycle

细胞复制的能力是许多生物过程的核心，单细胞 RNA-seq 允许在没有外部扰动的情况下研究调节细胞周期的机制。DeepCycle 是一种为数据集中的每个细胞分配一个角度（转录阶段）的方法，可以通过进一步分析将其与细胞周期阶段相关联。该方法基于 RNA 速度和自动编码器，能够推断数据中的潜在循环结构。DeepCycle 代表了一种研究单细胞 RNA-seq 数据集中细胞周期的可靠方法。
